# RMUS-2023
RoboMaster 2023 机甲大师高校 Sim2Real 挑战赛

## 常用命令
```bash
# 构建 Docker 镜像
scripts/build_image.sh
# 启动测试镜像
CLIENT_IMAGE=test scripts/launch.sh # bash # 是否启动 start.sh
# 同步代码到容器并编译
scripts/sync2container.sh
# 终止容器
scripts/halt.sh
```
